How It Works

1. TensorFlow.js

This app uses TensorFlow.js, a JavaScript library that brings machine learning to the browser. It allows us to run complex AI models directly in your web browser without needing a server.

2. MobileNet Model

We're using a pre-trained MobileNet model, which is a lightweight neural network designed for mobile and web applications. It can recognize 1000 different types of objects in images.

3. Image Processing

When you upload an image, here's what happens:

  • The image is converted into a tensor (a multi-dimensional array of numbers)
  • The image is resized to 224x224 pixels (the size MobileNet expects)
  • The pixel values are normalized to help the model process them better

4. Prediction Process

The model analyzes the processed image and provides probabilities for different object classes. We show you the top 5 predictions with their confidence scores.

5. Classes

The model can recognize 1000 different object categories, including:

  • Animals (dogs, cats, birds, etc.)
  • Objects (chairs, tables, phones, etc.)
  • Food items (fruits, vegetables, prepared dishes)
  • Vehicles (cars, bikes, planes)
  • And many more!
